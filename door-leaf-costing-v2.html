📋 Price List 导入格式

系统自动识别列名，支持 CSV 和 Excel。


必须包含（任意一个）：

• 材料名：Drawing Code / Material / Description / Name

• 单价：Cost / Price / Unit Price


可选列（有则自动带入）：

• Item Code / Code — 料号

• Unit / UOM / remark — 单位


💡 直接从 Google Sheet Price List 导出 CSV 即可


匹配逻辑：

Paste 进 BOM 表的材料名称，系统自动与 Price List 的 Drawing Code 做模糊匹配，匹配成功自动填入 Item Code 和单价。
